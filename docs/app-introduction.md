# 记事本 — 应用介绍

## 概述

记事本是一款轻量、精美的多用户笔记管理应用，适合部署在飞牛NAS上，供家庭或团队内部使用。基于 Go + Vue 3 构建，单二进制文件部署，资源占用极低。

## 功能特性

### 笔记管理
- **富文本编辑**：基于 TipTap 编辑器的所见即所得编辑体验，支持标题、加粗、斜体、下划线、删除线、引用、代码块、有序/无序列表、文字颜色、图片插入等排版功能
- **标签分类**：为笔记添加标签，通过标签快速筛选和归类
- **笔记搜索**：全局搜索笔记标题和内容，快速定位
- **排序切换**：支持按最近修改、最近创建、标题字母序排列
- **自动保存**：编辑内容 3 秒自动保存，无需手动频繁操作，支持 Ctrl+S 快捷键
- **查看/编辑模式**：点击笔记进入详情阅读模式，点击编辑按钮进入编辑，阅读编辑分离

### 多用户支持
- 用户注册、登录、退出
- 每个用户的数据完全隔离，互不可见
- 通过安全问题重置密码
- 首个注册用户自动成为超级管理员

### 管理员功能
- 用户管理：查看用户列表、创建用户、重置密码、设置角色
- 系统配置：管理站点标题、是否允许注册等配置项
- 终端命令辅助：支持 `find-admin`、`recover-admin`、`list-users` 等 CLI 命令

### 界面体验
- **暗色模式**：一键切换明暗主题，护眼舒适
- **响应式设计**：完美适配 PC 端和移动端，随时随地记录
- **仪表盘**：登录后展示总笔记数、标签数、最近笔记概览
- **笔记列表页**：卡片式布局展示笔记，支持搜索筛选分页

## 适用场景

- **家庭笔记共享**：家庭成员各自记录，数据隔离互不干扰
- **团队知识库**：小型团队内部记录技术文档、会议纪要
- **个人知识管理**：个人笔记、灵感记录，随时随地通过浏览器访问
- **NAS 用户的轻量替代**：相比庞大的笔记系统，记事本占用资源极少，部署简单

## 技术特性

- **纯 Go 后端**：单二进制文件，无外部依赖，CGO_ENABLED=0 编译，部署极其简单
- **嵌入式 SQLite**：无需额外数据库服务，数据存储在本地文件，备份迁移方便
- **前端嵌入二进制**：Vue 3 前端构建后嵌入 Go 二进制，无 Node.js 运行时依赖
- **数据库自动迁移**：升级版本时自动执行数据迁移，无需手动操作
- **JWT 认证**：基于 Token 的身份认证，安全可靠

## 部署信息

| 项目 | 说明 |
|------|------|
| 默认端口 | 8904 |
| 数据目录 | /app/data（Docker）或 ./data（本地） |
| 访问方式 | 浏览器打开 http://<NAS_IP>:8904 |
| 资源占用 | 极低，适合长期运行在 NAS 上 |

## 首次使用

1. 安装完成后打开应用
2. 注册第一个账号（自动成为管理员）
3. 开始创建笔记
